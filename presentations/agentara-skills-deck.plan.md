# agentara-skills-deck.plan

## Deck Overview
- **Topic:** Agentara Skills — AI 开发者的开源技能库
- **Audience:** AI 开发者、Claude Code 用户、技术团队
- **Tone:** 现代科技感、轻盈、清晰
- **Style Mode:** dark glassmorphism (macOS 毛玻璃风格)
- **Language:** 中文

## Style Direction
- **Background:** 深色模糊渐变背景（蓝紫/青色渐变）
- **Primary accent:** 半透明白色面板 + 柔和白色边框
- **Typography:** Geist / Inter 中等字重，简洁几何无衬线
- **Image treatment:** 毛玻璃卡片层叠于彩色模糊背景之上

## Layout Sequence
1. `hero-title` — 开场标题页
2. `split-2col` — 项目概述（文字 + 视觉）
3. `3-card` — 三大技能分类
4. `split-2col` — 热门技能展示
5. `big-number` — 项目数据（Stars、Fork、Skills 数量）
6. `closer` — 结束页 / CTA
